3. Jelaskan fungsi tag HTML pada Table: thead, tbody, tr, th serta td!
> thead >> menunjukkan sebuah table head atau bagian kepala sebuah tabel (table element)
> tbody >> menunjukkan sebuah table body atau bagian badan sebuah tabel (table element)
> tr    >> menunjukkan table row atau baris sebuah table
> th	>> untuk membuat sebuah heading dari tabel
> td	>> menunjukkan table data atau bagian data dari sebuah tabel
